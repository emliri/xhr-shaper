XHR-shaper v2 demo

github.com/tchakabam/xhr-shaper

XHR-shaper is a network conditioner for the browser emulating bandwidth and latency.

  • It can be used in test environments to check systems behaviours against real-world.
  • It may be used on top of any other standard compliant XHR implementation
  • It also allows to have a global cache over all previous XHR traffic.
  • It can be used as a global shim over the native XMLHttpRequest object or as a local module.

Read more about it here. Check the Github repository for full docs.
